- 采用纯静态单页架构，以 `index.html` 为入口，通过内联脚本与外部 `i18n.js`、`style.css` 协同工作。
- `i18n.js` 封装了基于 `localStorage` 与 URL 参数的区域设置解析逻辑，并通过 DOM 查询动态替换文本与属性，支持 EN/PT-BR。
- `vercel.json` 定义了零构建（`buildCommand: null`）的托管策略，针对静态资源（CSS/SVG）配置了差异化的缓存头与安全响应头（如 `X-Frame-Options: DENY`）。
- 页面内部通过 CSS 类名切换（如 `.mockup-view-active`）实现应用界面的多视图（聊天、终端、编辑器）交互模拟，不依赖重型前端框架。
