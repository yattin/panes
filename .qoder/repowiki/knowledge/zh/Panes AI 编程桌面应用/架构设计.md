- **混合架构**：采用 Tauri 2 框架，前端为 React + Vite 单页应用，后端为 Rust 核心库，通过 Tauri IPC 协议进行双向通信。
- **工作区驱动**：以“工作空间（Workspace）”为核心上下文，统一管理 Git 仓库、文件系统及 AI 对话线程的状态持久化（SQLite）。
- **多引擎适配**：后端实现统一的引擎抽象层，支持 Claude Code、Codex、OpenCode 等多种 AI 代理的插件式集成与动态切换。
- **Sidecar 协同**：通过 Node.js Sidecar 进程处理特定的 AI SDK 通信任务，由 Rust 后端负责进程生命周期管理与安全隔离。
